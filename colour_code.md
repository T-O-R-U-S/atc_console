| White  | `ff`       |
| Gray | `0f` |
| LightGray | `1f` |
| DarkRed | `0a` |
| Red | `1a` |
| DarkGreen   | `0b`       |
| Green   | `1b`       |
| DarkBlue   | `0c`       |
| Blue   | `1c`       |
| Cyan | `2c` |
| Yellow | `ab` |
| Orange | `fa` |
| Brown | `30` |
| Purple | `ac` |
| Pink | `bf` |
| Transparent | `99` |
